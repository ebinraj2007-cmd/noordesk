Acme Store — demo site

This is a sample website page. The chat bubble in the bottom-right corner is the NoorDesk web widget. Type a message and it appears on your NoorDesk dashboard as a web message, triaged like any other channel.

To put it on your real site, add one line before </body>:

<script>window.NOORDESK_CORE="http://127.0.0.1:8000";</script>
<script src="widget.js"></script>
